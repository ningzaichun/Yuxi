### 📝 变更内容
将量程科技提供的正式 Logo 原图裁切并生成 512×512 主 Logo 与 128×128 favicon，分别接入 Web 应用和 VitePress 文档站。默认组织 Logo、头像、登录页回退资源及浏览器图标统一改用 PNG 文件，并删除临时 K-AI SVG 与旧黄色 favicon。

### 🔍 变更原因
项目此前使用的是临时品牌标识，需要替换为量程科技确认的正式 Logo，并保证登录页、首页、侧边栏、浏览器标签和文档站使用统一品牌资源。

### ⚠️ 注意事项
正式 Logo 来源于用户提供的 PNG，仅进行了确定性裁切、缩放和 PNG 优化，没有重新绘制或改变品牌图形。依赖自定义组织配置的部署环境如覆盖了 `organization.logo` 或 `organization.avatar`，仍会优先使用其自定义路径。

### 🧪 测试建议
- 执行 `pnpm --dir web run build`，确认前端生产构建成功。
- 启动前后端后检查登录页、首页、侧边栏及浏览器 favicon 的 Logo 显示。
- 构建文档站后检查导航 Logo 与文档站 favicon。
