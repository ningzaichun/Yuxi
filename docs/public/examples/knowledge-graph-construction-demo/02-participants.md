# 滨江科创中心一期工程参建方

> 演示数据声明：本文档中的项目、组织、人员、合同、事件和数值均为虚构，仅用于 Yuxi 知识图谱功能测试。

滨江科创中心一期工程由江州滨科置业有限公司组织建设。建设单位项目负责人周明远任职于江州滨科置业有限公司，担任建设单位项目负责人，负责重大建设事项协调，并参与该项目的例会与验收决策。

江州建筑设计研究院有限公司承担项目总体设计。建筑专业负责人林岚任职于江州建筑设计研究院有限公司，担任建筑专业负责人，重点负责研发办公楼和实验楼的建筑设计协调。

华东新城建设集团有限公司承担项目施工总承包。陈启航任职于华东新城建设集团有限公司，担任项目经理并参与滨江科创中心一期工程，负责施工组织和现场履约。赵凯同样任职于华东新城建设集团有限公司，担任安全负责人，负责现场安全管理。

安衡工程咨询有限公司承担项目监理。孙若川任职于安衡工程咨询有限公司，担任总监理工程师，负责监督滨江科创中心一期工程的质量、安全和进度。

江州钢铁供应链有限公司和清源机电设备有限公司作为专业供应单位参与项目，分别提供钢筋材料和暖通设备。
