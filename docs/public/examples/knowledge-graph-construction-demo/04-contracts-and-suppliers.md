# 合同与供应关系说明

> 演示数据声明：本文档中的项目、组织、人员、合同、事件和数值均为虚构，仅用于 Yuxi 知识图谱功能测试。

江州滨科置业有限公司与华东新城建设集团有限公司签订施工总承包合同，合同编号为 BJ-KC-ZCB-2025-001。该合同覆盖滨江科创中心一期工程以及研发办公楼、实验楼和地下车库的施工总承包工作。

华东新城建设集团有限公司与江州钢铁供应链有限公司签订钢筋采购合同，合同编号为 BJ-KC-GJ-2025-017。江州钢铁供应链有限公司根据该合同向项目供应 HRB400E 钢筋，材料用于研发办公楼、实验楼和地下车库的主体结构。

华东新城建设集团有限公司与清源机电设备有限公司签订暖通设备采购合同，合同编号为 BJ-KC-NT-2026-006。清源机电设备有限公司根据合同向实验楼供应磁悬浮冷水机组。

项目使用的 C40 商品混凝土由江州恒基建材有限公司供应，主要用于三个单体的主体结构。地下车库施工使用的塔式起重机由华东新城建设集团有限公司统一组织进场和使用。
