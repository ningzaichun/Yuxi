# 建筑单体与关键技术说明

> 演示数据声明：本文档中的项目、组织、人员、合同、事件和数值均为虚构，仅用于 Yuxi 知识图谱功能测试。

滨江科创中心一期工程在研发办公楼和实验楼的设计、施工协同中使用 BIM 协同平台。研发办公楼采用装配式施工技术，主体结构使用 C40 商品混凝土和 HRB400E 钢筋。研发办公楼核心筒施工由华东新城建设集团有限公司负责，建筑设计协调由林岚负责。

实验楼对楼板振动和机电环境控制要求较高，主体结构同样使用 C40 商品混凝土和 HRB400E 钢筋。实验楼暖通系统选用磁悬浮冷水机组，由清源机电设备有限公司供应。实验楼的建筑设计协调由林岚负责。

地下车库采用防水混凝土施工技术，结构施工使用 C40 商品混凝土和 HRB400E 钢筋。地下车库施工阶段配置塔式起重机进行材料垂直运输，现场施工组织由华东新城建设集团有限公司负责。

研发办公楼、实验楼和地下车库均位于滨江科创中心一期工程范围内。
